★これは何？
　アルゴンについて、リアルタイム古典分子動力学シミュレーションを行うコードです。
　このコードは、ロボ太（@kaityo256）様の、分子動力学法ステップ・バイ・ステップ
　（ https://github.com/kaityo256/mdstep ）を参考にさせて頂いています。
　ビルドには、以下のライブラリが必要です。
　・Boost C++ Libraries
　・Eigen
　・Intel® Threading Building Blocks (Intel® TBB)

★更新履歴
　2018/8/3    ver.0.1　公開。
